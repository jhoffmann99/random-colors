Random Colors

This simple and easy to use tool helps finding some random colors that match together.

Features

  • Press SPACE to spin the color wheel
  • Click on a box to toggle the color lock. Locked colors are annotated with a solid red border
  • Copy a hex color code by clicking on a box
  • Press "R" key to reset all color locks
  • Press "L" key to disable/enable the darkmode
  • Use the below input field to set the size of your color palette
  • Press ESCAPE to show this popup
